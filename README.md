# RAO BUS Service / Rao Tour and Travel

Premium transport and travel agency website UI for RAO BUS Service, Farrukhnagar, Gurugram, Haryana.

## Overview

This project is a modern React + Vite website for a local Indian transport business offering:

- Bus rental
- Tempo traveller booking
- Private car travel
- Pilgrimage tours
- Hill station tours
- Wedding and group travel booking

The design uses a cinematic dark travel theme with glassmorphism, animated sections, responsive layouts, gallery lightbox, testimonials, booking form UI, and premium transport-focused visuals.

## Tech Stack

- React
- Vite
- CSS animations
- Lucide React icons

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
